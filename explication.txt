 Agent IA d'Analyse de Données (Google Sheets + n8n)
 Description du Projet
Ce projet consiste en la conception et l'implémentation d'un Agent Intelligent autonome via l'outil d'automatisation n8n. L'objectif est de transformer une feuille de calcul Google Sheets statique (ma liste de livres) en une base de connaissances interactive capable de répondre à des requêtes complexes en langage naturel.

Fonctionnement Technique du Workflow
Le workflow est structuré autour de quatre composants majeurs qui communiquent entre eux :

1. L'Interface de Chat (Trigger)
Le point d'entrée est un nœud Chat Trigger. Il permet d'envoyer des messages textuels à l'agent. C'est ici que l'utilisateur pose ses questions (ex: "Quels livres d'aventure ai-je dans ma liste ?").

2. Le Cerveau : AI Agent (Nœud Central)
C'est le composant le plus important. Contrairement à un workflow classique qui suit un chemin linéaire, l'Agent IA utilise une logique de raisonnement :

Il analyse la question de l'utilisateur.

Il décide s'il a besoin d'utiliser un outil externe (Google Sheets) pour répondre.

Il synthétise les données récupérées pour donner une réponse humaine.

3. Le Modèle de Langage (LLM) : OpenAI
J'ai intégré le modèle de OpenAI pour fournir l'intelligence de compréhension. Il permet à l'agent de comprendre les nuances, les fautes d'orthographe ou les demandes indirectes.

4. L'Outil Métier : Google Sheets Tool
C'est le "bras armé" de l'agent. J'ai configuré un outil spécifique qui permet à l'IA de :

Lire l'intégralité ou des lignes spécifiques du fichier.

Analyser les colonnes (Titre, Auteur, Genre, etc.).

Extraire uniquement l'information pertinente pour répondre à la question.

Gestion de la Mémoire
Le workflow inclut un nœud Window Buffer Memory. Cela permet à l'agent de se souvenir des messages précédents.

Exemple : Si je demande "Qui est l'auteur du premier livre ?", puis "A-t-il écrit autre chose ?", l'agent sait que "il" se rapporte à l'auteur cité précédemment.

Capacités Clés
Extraction de données : Retrouver instantanément n'importe quel livre sans ouvrir le fichier.

Analyse de contenu : L'IA peut résumer les thématiques présentes dans le Google Sheets.

Recommandation intelligente : Basé sur la liste existante, l'IA peut suggérer quel livre lire ensuite.
